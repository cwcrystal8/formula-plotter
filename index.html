Formula Plotter

Introduction

This web component creates an interface for the user to enter in a single-variable formula and automatically plot that formula over a customizable domain.

Demos

Simple Graph

Here is a simple example where the variable is x and the formula is f(x) = x.

HTML

<formula-plotter id='demo-1' var-name='x'></formula-plotter>

Javascript

let plot = document.querySelector('#demo-1');
plot.formula = 'x';

Output

Complex Formula

Here is a more complicated formula to demonstrate the many operations supported by the API.

HTML

<formula-plotter id='demo-2' var-name='x'></formula-plotter>

Javascript

let plot = document.querySelector('#demo-2');
plot.formula = '0.25x^3 + x^x - (0.2 * x) + 5!/120';

Output

Domain Resizing

Here is an example of resizing beyond the default domain range.

HTML

<formula-plotter id='demo-3' var-name='x'></formula-plotter>

Javascript

let plot = document.querySelector('#demo-3');
plot.formula = '0.25x^3';
plot.xMax = 500;
plot.xMin = -25;

Output

Step Adjustment

Here is an example of adjusting the step size to plot more datapoints for a smoother curve.

HTML

<formula-plotter id='demo-4' var-name='x'></formula-plotter>

Javascript

let plot = document.querySelector('#demo-4');
plot.formula = '0.25x^3';
plot.step = 0.5;

Output

Number of Datapoints Adjustment

Similar to the previous example, this demo will set the number of datapoints (instead of the step size) to adjust the smoothness of the plot.

HTML

<formula-plotter id='demo-5' var-name='x'></formula-plotter>

Javascript

let plot = document.querySelector('#demo-5');
plot.formula = '0.25x^3';
plot.setNumDatapoints(50);

Output
