JS Documentation

JS Quick Start

In order to access the formula-plotter element in JavaScript, you can run the following code to get its DOM object: 

let plotter = document.querySelector("formula-plotter");

Accessors

The following variables are accessible in JavaScript:

  • varName (str): the value of the element's var-name attribute. This reflected attribute-property can be changed in JS as such: 
    plotter.varName = 'y';
  • formula (str): the current formula being plotted.
  • xMin (number): the first value to be plugged into the formula and plotted. Default value of 0.
  • xMax (number): the last value to be plugged into the formula and plotted. Default value of 100.
  • step (number): the step size taken between each datapoint. Default value of 100. For example, a step size of 1 would results in datapoints (xMin, f(xMin)), (xMin + 1, f(xMin + 1)), and so on.

Methods

The formula-plotter has the following methods:

  • get varName(): returns the variable used in the formula as a string.
  • set varName(v): sets the variable name to v and reloads the graph.
  • setFormula(f): sets the formula to f and reloads the graph
  • setXMin(x): sets the lower bound of the domain to x and reloads the graph
  • setXMax(x): sets the upper bound of the domain to x and reloads the graph
  • setStep(step): sets the step size to step and reloads the graph
  • setNumDatapoints(n): takes in an integer, n, and changes plotter.step so that there are n datapoints generated between plotter.xMin and plotter.xMax.

Events

The formula plotter fires an event with the name plotted whenever it is done plotting the graph and the event object returned contains the following properties:

  • formula (str): the formula used to plot the graph
  • xMin (number): the lower bound of the domain
  • xMax (number): the upper bound of the domain
  • xValues (array): the x-coordinates of the plotted datapoints
  • yValues (array): the y-coordinates of the plotted datapoints
