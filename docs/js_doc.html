JS Documentation

JS Quick Start

In order to access the formula-plotter element in JavaScript, you can run the following code to get its DOM object: 

let plotter = document.querySelector("formula-plotter");

Accessors

The following variables are accessible in JavaScript:

  • varName (str): the value of the element's var-name attribute. This reflected attribute-property can be changed in JS as such: 
    plotter.varName = 'y';
  • formula (str): the current formula being plotted. This property can be changed in JS as such: 
    plotter.formula = 'y ^ 2';
  • xMin (number): the first value to be plugged into the formula and plotted. This property can be changed in JS as such: 
    plotter.xMin = -5;
  • xMax (number): the last value to be plugged into the formula and plotted. This property can be changed in JS as such: 
    plotter.xMax = -5;
  • step (number): the step size taken between each datapoint. For example, a step size of 1 would results in datapoints (xMin, f(xMin)), (xMin + 1, f(xMin + 1)), and so on. This property can be changed in JS as such: 
    plotter.step = 0.5;

Methods

The formula-plotter has one method: setNumDatapoints. setNumDatapoints(n) takes in an integer, n, and changes plotter.step so that there are n datapoints generated between plotter.xMin and plotter.xMax

Events

Whenever the formula, x-min, or x-max values are changed, either by the user in the UI or by the developer in JS, the plot/graph will update to reflect the new parameters.

There are no additional events of note.
