Attributes

The ‹formula-plotter› element has one attribute called var-name that determines the variable name in the provided formula. var-name can be any sequence of letters, numbers, symbols, etc. but the provided formula must be valid and able to be evaluated after all instances of var-name have been replaced with a number.

For example, the following code would create a formula plotter object that could only take in formulas using x as a variable: 

‹formula-plotter var-name='x'›
    ...
‹/formula-plotter›

Children

Each ‹formula-plotter› element has one child attached to its shadow root with the following structure: 

‹formula-plotter var-name='x'›
    ‹div id='inputs' part='inputs'›
        ‹label› Formula
            ‹input type='text' id='formula'›
        ‹/label›
        ‹label› X-Min
            ‹input type='number' id='x-min'›
        ‹/label›
        ‹label› X-Max
            ‹input type='number' id='x-max'›
        ‹/label›
    ‹/div›

    ‹div id='graph' part='graph'›‹/div›
‹/formula-plotter›
