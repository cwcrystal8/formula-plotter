CSS Documentation

Parts

The ‹formula-plotter› element has two parts called inputs and graph. Users can use these parts to style the input fields and the graph div, respectively.

Examples

To change the width of the graph too 500px, you can include the following CSS in your project: 

::part(graph) {
    width: 500px
}

To change the inputs so that they are vertically aligned, you can include the following CSS in your project: 

::part(inputs) {
    display: flex;
    flex-flow: column
}
